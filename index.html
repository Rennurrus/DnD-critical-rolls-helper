Critical Rolls Helper

Один автономный HTML-файл для телефона. Работает офлайн после открытия.

Eсли включён риск-фильтр, приложение не заставляет делать перебросы, а детерминированно remap’ит выпавшую таблицу в допустимый диапазон текущего tier.
Как работает фильтр риска

Raw table = обычный номер таблицы по формуле (к10 − 1) × 6 + к6.

Если выбран режим Авто, tier определяется по уровню героев: 1–4 → tier 1, 5–8 → tier 2, 9–12 → tier 3, 13+ → tier 4.

Если raw-таблица не входит в разрешённый диапазон текущего tier, приложение без перебросов переносит её внутрь нужного диапазона. Так офлайн за столом пользоваться сильно проще.
